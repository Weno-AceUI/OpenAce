Style Guide & Component Library

Glass Card (`.ace-card-glass`)

This is content inside a glass card. It has a frosted background and rounded corners.

Usage:

<div class="ace-card-glass p-6"> Your content here... </div>

Buttons

Primary Button (`.ace-button-primary`)

<button class="ace-button-primary">Primary Action</button>

Danger Button (`.ace-button-danger`)

Circular Button (`.ace-button-circular`)

<button class="ace-button-circular">1</button> <button class="ace-button-circular w-16 h-16 text-2xl">Big</button>

Note: Circular buttons default to w-14 h-14 text-xl. You can override size and font size with Tailwind utilities.
